회원관리 시스템 요구사항 정리

1. 회원 등록 및 프로필 관리
•	부모/보호자 계정 연동 (COPPA 준수)
o	미성년자 대상으로, 모든 회원 등록은 부모/보호자 이메일 인증과 동의서(Terms & Condition Checkbox) 획득이 필수
o	부모 계정 하나에 여러 자녀 프로필을 연결할 수 있도록 설계
등록 시 이름, 나이, 연락처, 주소, 건강 정보(알레르기 등)를 수집
	COPPA에 따라 13세 미만 아동의 개인정보는 최소화
•	자동 승인/거부 워크플로
o	등록 신청 시 관리자가 검토할 수 있는 대시보드 제공.
o	AI 기반으로 중복 등록이나 유효성(나이 범위 확인)을 확인해 자동 승인/알림
•	프로필 업데이트 자동화
o	회원 또는 부모가 앱/웹에서 프로필을 업데이트할 때, 변경 내역을 자동 로그하고 관리자에게 알림
o	연간 재등록 알림
•	추가 자동화 기능
o	회원 탈퇴 시 자동 데이터 삭제(법적 요구사항 준수)와 환불 처리 연동
	Policy에 따라 변동 사항 발생
o	회원 등급 자동 업그레이드
	나이 증가에 따라 청소년 그룹으로 이동
	Level또는 Age 별 Grouping
	Private Lesson Group
2. 출석 관리 (모바일 앱 중심) – MVP 2
•	앱 기반 체크인/아웃
o	QR 코드 스캔, GPS 위치 확인, 또는 블루투스 비콘을 통해 출석 자동 기록
o	앱은 iOS/Android 지원, 오프라인 모드(나중에 동기화) 제공
o	멤버십 카드에 QR & 사진을 프린트 하여 도장에 비치
	Check In 완료 Notification 은 SMS
	iPad 화면에 QR 스캔 | 검색 을 넣어 스캔 선택 시 카메라를 연동하여 스캔 가능하도록 설정
	스캔 완료 된 아이디의 경우 참석자 명단에 사진 포함하여 디스플레이
•	Alert Rule 설정 및 자동화
o	관리자가 규칙을 커스텀 설정 가능
	7일 이상 미출석 시 출석 독려"이메일
	미출석 기간 3일/7일/14일 임계값 설정
	Late Payment 시 "결제 알림" 이메일 
	결제 지연 5일/10일 후 자동 알림
o	자동 이메일 템플릿 개인화(회원 이름 삽입), 첨부 파일(출석 보고서 등) 지원
•	추가 자동화 기능 – MVP N(?)
o	AI 기반 출석 예측
	과거 데이터로 미출석 위험 회원 식별 후 사전 알림
	그룹 출석 관리(클래스별 일괄 체크인)
	출석 데이터 실시간 대시보드(관리자용)로 출석률 통계 자동 생성
•	부모 드랍오프 시 출석 체크 방안
o	자동화된 무인 체크인 스테이션
	Kiosk 또는 iPad with stand
	직원이 Validate 후 최종 승인(이중 체크)
	카메라 연동으로 얼굴 인식 또는 NFC 태그 (아이 배지)로 자동 로그
o	부모 앱을 통한 원격 드랍오프 예약
	부모가 앱을 통해 사전 드랍오프 시간 예약하고 도착시 GPS 트리거로 시스템에 자동 알림 후 직원이 앱으로 이중 체크 후 Validate
	체크인 완료 후 부모에게 Push Notification 발송
o	직원/보조 체크인 프로세스
	직원이 모바일 앱으로 바코드/QA 스캔 또는 이름 검색 후 체크인
o	안전 강화 기능
	드랍오프 후 5분 내 미확인 시, 시스템 자동 Alert(직원 대시보드 팝업 및 부모 알림)
o	법적/안전 고려 사항
	모든 드랍오프 로그에 Time Stamp, 위치 데이터, 직원 ID 기록
	단, 부모 동의에 한해 자동화 가능
o	추가 자동화 기능
	드랍오프 패턴 분석
	AI로 빈번한 드랍오프 부모 식별 후 맞춤 알림 템플릿 제공
3. 알림 및 이메일 자동화
•	자동 이메일 발송 시스템
o	Alert Rule 기반으로 트리거
	미출석, Late Payment
o	이메일 서비스
	SendGrid 또는 AWS SES 통합으로 안정성 확보하고 발송 로그 저장 및 재발송 기능 추가
•	심사 정보 이메일 follow-up
o	심사 과정에서 자동 follow-up 이메일 
	심사 서류 제출 요청 후 3일 경과 시 리마인더
	템플릿 편집기 제공
•	다채널 알림 확장
o	이메일 외 SMS - Twilio
o	푸시 알림(앱 내) 또는 부모 포털 알림
o	추가 기능으로 이벤트 알림(클래스 시작 1시간 전 자동 푸시)
•	추가 자동화 기능
o	에스컬레이션 룰
	2회 알림 무응답 시 관리자 태스크 생성
	 알림 효과 분석(오픈률/클릭률 자동 보고서 생성)으로 캠페인 최적화
4. 온라인 결제 및 재무 관리
•	심사비 온라인 결제
o	Stripe 또는 PayPal 통합으로 신용카드/ACH 결제 지원
o	결제 페이지에서 심사비 자동 계산(할인 적용 가능)
영수증 자동 이메일 발송
•	멤버십 비용 관리
o	월/연간 구독 모델
o	멤버십 Hold/Suspend 기능
o	Late Payment 자동 감지
	블랙리스트 관리
o	환불 정책 자동화 (도장 Policy 준수)
	탈퇴 시 비례 환불 계산
	메뉴얼 기능으로 추가 예정
o	Grace Period 기능 추가 (열흘정도 예상)
•	추가 자동화 기능
o	자동 청구(Recurring Billing)와 만료 알림(결제 만료 7일 전 이메일)
o	세금 계산(미국 주별 세율 자동 적용)
o	재무 보고서 자동 생성(월별 수익/미납 목록 엑셀 export)
5. 웹사이트 기능 (관리자 및 사용자 포털)
•	관리자 대시보드
o	회원 목록, 출석 로그, 알림 히스토리, 결제 내역 실시간 보기. 검색/필터 기능(나이, 그룹별)
•	부모/회원 포털
o	프로필 보기, 출석 기록 조회, 결제 내역 확인, 심사 Form Submit. 모바일 반응형 디자인
•	보안 및 컴플라이언스 
o	SSL 암호화, 2FA(두 단계 인증), 데이터 암호화. COPPA 준수 위해 13세 미만 데이터는 부모 동의 없이 수집 금지
o	감사 로그(모든 변경 추적)
•	추가 자동화 기능
o	백업 자동화(일일 데이터 백업), API 통합(외부 CRM 또는 캘린더 앱 연동)
AI 챗봇으로 문의 자동 응답
•	소셜 미디어 
6. 보고서 및 분석 기능 (추가 자동화 중심)
•	자동 보고서 생성
o	주간/월간 출석 보고서(PDF/Excel export), 
o	회원 유지율 분석(예: 탈퇴 이유 예측 AI)
o	Alert Rule 효과 보고(발송 후 출석 증가율 측정)
•	데이터 시각화
o	차트/그래프(예: 나이별 회원 분포, 출석 추이)로 대시보드 제공
•	추가 자동화 기능
o	머신러닝 기반 인사이트(예: "저출석 회원 그룹화" 후 타겟 마케팅 이메일 자동 발송)
7. 시스템 통합 및 확장성
•	통합
o	이메일(MailGun), 결제(Stripe), SMS(Twilio), 캘린더(Google Calendar) 등 외부 API 연동
•	모바일 앱 추가 기능
o	오프라인 출석 기록, 사진 업로드(활동 증빙), 부모-자녀 메시징
•	스케일링 고려
o	사용자 증가 시 서버 자동 스케일링(클라우드 기능 사용)
o	다국어 지원(영어 기본, 스페인어, 한국어 옵션)
